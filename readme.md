1. **דף בית** - הצגת העסק והשירותים
2. **גלריה** - 53 תמונות באיכות גבוהה עם אפשרות הגדלה
3. **שירותים** - פירוט כל הטיפולים המוצעים
4. **צור קשר** - טופס יצירת קשר ופרטי התקשרות

- **אפשרות הגדלת תמונות** - לחיצה על תמונה מציגה אותה בגודל מלא
